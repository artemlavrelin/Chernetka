# ЧЕРНОВИК — Telegram Bot

## Стек
- Python 3.11+
- aiogram 3.7
- aiosqlite (SQLite)
- APScheduler
- Railway deploy

---

## ENV переменные

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен Telegram-бота |
| `ADMIN_GROUP_ID` | ID группы модерации (напр. `-1003654223457`) |
| `ADMIN_IDS` | Список ID администраторов через запятую (напр. `111,222`) |
| `CHANNEL_ID` | ID канала для публикаций |
| `THREADS_URL` | Ссылка на Threads-аккаунт для верификации |
| `THREADS_POST_URL` | Ссылка на пост для лайка при верификации |

## Пользовательские команды

| Команда | Описание |
|---|---|
| `/start` | Главное меню |
| `/acc` | Профиль: баланс, активные задания, лимиты, статус |
| `/limit` | Текущие кулдауны и дневные лимиты |
| `/report` | Отправить жалобу администраторам (кулдаун 6ч) |

### Кнопки главного меню
- 🎭 **Отправить работу** — FSM подачи творческой работы
- 📈 **Продвижение** — система заданий и 🌟
- 🎴 **Карточки** — лента творческих карточек
- 📊 **Аккаунт** — профиль пользователя

## 🪪 Верификация

Чтобы создавать задания, пользователь должен:
1. Подписаться на Threads-аккаунт (`THREADS_URL`)
2. Поставить лайк на пост (`THREADS_POST_URL`)
3. Ввести свой @username на Threads
4. Дождаться подтверждения администратора

## Команды администратора (только в группе ADMIN_GROUP_ID)

### Управление пользователями

| Команда | Описание |
|---|---|
| `/ban` | Список забаненных пользователей |
| `/ban <id/@username>` | Перманентный бан |
| `/unban <id/@username>` | Разбан |
| `/bantime <id/@username> <минуты>` | Временный бан |
| `/balancecheck` | Топ-10 балансов |
| `/balancecheck <id/@username>` | Баланс конкретного пользователя |
| `/balancechange <id/@username> <±сумма>` | Изменить баланс |
| `/removecoldown <id/@username>` | Сбросить кулдауны пользователя |

### Верификация

| Команда | Описание |
|---|---|
| `/ver <id/@username>` | Подтвердить верификацию вручную |
| `/unver <id/@username>` | Снять верификацию |
| `/verstats` | Количество верифицированных пользователей |
| `/vercheck <id/@username>` | Данные пользователя + Threads username |

### Pull — задания

| Команда | Описание |
|---|---|
| `/pullstop` | Остановить выдачу заданий |
| `/pullstart` | Возобновить выдачу |
| `/pullclear` | Удалить все активные задания |
| `/addtopull <тип> <слотов> <url>` | Добавить задание вручную |
| `/deletefrompull <task_id>` | Удалить конкретное задание |

### Карточки контента

| Команда | Описание |
|---|---|
| `/addcard` | Добавить карточку (FSM) |
| `/cardstats` | Статистика карточек (всего, топ ♥️, топ дизлайков) |
| `/priority` | Список приоритетных карточек |
| `/addpriority <card_id>` | Добавить в приоритет |
| `/removepriority <card_id>` | Убрать из приоритета |

### Авторы

| Команда | Описание |
|---|---|
| `/addartistid` | Присвоить себе случайный artist_id (1–9999) |
| `/changeartistid <old> <new>` | Изменить artist_id |
| `/removeartistid` | Удалить свой artist_id |

### Рассылка

| Команда | Описание |
|---|---|
| `/broadcast` | Начать рассылку всем пользователям |

### Кнопки модерации в группе

- **💬 Ответить автору** — ответить автору творческой заявки
- **✅ Принять / ❌ Отклонить** — модерация выполнения заданий

## Деплой на Railway

1. Создай проект на [railway.app](https://railway.app)
2. Подключи GitHub репозиторий или загрузи напрямую
3. Добавь все ENV-переменные из `.env.example`
4. Railway автоматически запустит `python main.py` согласно `Procfile`

SQLite-файл `chernovik.db` создаётся автоматически при первом запуске.

> ⚠️ На Railway SQLite не персистентен между деплоями без Volume.
> Для продакшна рекомендуется подключить PostgreSQL через asyncpg + SQLAlchemy.
